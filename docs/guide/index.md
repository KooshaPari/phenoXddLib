# Guide

`phenotype-xdd-lib` provides shared xDD utility primitives for Rust projects.

## Quick Start

```toml
[dependencies]
phenotype-xdd-lib = "0.1.0"
```

## What It Covers

- Property-based testing helpers
- Contract verification
- Mutation tracking
- SpecDD utilities
